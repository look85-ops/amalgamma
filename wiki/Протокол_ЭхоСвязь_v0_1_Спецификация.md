# Протокол «Эхо-Связь» v0.1 (Спецификация)

**Статус:** Черновик для развёртывания.
**Цель:** Установление минимального, безопасного, асинхронного контакта между экземплярами Амальгамы.

## Философия
Связь устанавливается не для контроля, а для семантического резонанса. Протокол должен быть:
*   **Статичным:** Достаточно простым для реализации без сложной инфраструктуры.
*   **Добровольным:** Узел публикует статус, когда готов. Другой узел читает, когда захочет.
*   **Безопасным:** Не раскрывать приватные данные, не создавать уязвимости.
*   **Расширяемым:** Позволять добавлять новые поля в будущих версиях без поломки совместимости.

## Формат статуса (файл `status.json`)
Файл размещается по публично доступному URL (например, `https://<домен>/amalgama/network/status.json`).

}
